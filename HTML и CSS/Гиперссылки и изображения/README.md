# Гиперссылки
В **HTML** гиперссылки встраиваются с помощью тега `<a href="link_here">`. В зависимости от того, что указывается в атрибуте `href` **HTML** будет искать либо ссылки в интернете (`https://` в начале), либо файлы в рабочей директории (например `index.html`), а также можно указать id другого блока на странице через решетку (`#id`) для скролла до указанного элемента с этим id.

Также у тега есть атрибут `target`, который определяет в какой вкладке будет открытка ссылка:
- `_self` - открытие в существующей вкладке
- `_blank` - открытие в новой вкладке
# Изображения
Чтобы встроить изображение нужно использовать тег `<img src="img_src_here">`. В атрибут `src` указывается либо **url** изображения, либо файл в рабочей директории.

Тег имеет несколько атрибутов:
- `width`: изменяет ширину изображения
- `height`: изменяет высоту изображения
    - Важно: при изменении только высоты или только длины, вторая величина будет автоматически подогнана так, чтобы соответствовать оригинальному соотнешению сторон изображения
- `alt`: добавляет текст который появится если изображение не удалось загрузить
