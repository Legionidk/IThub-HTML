# Позиционирование в HTML с помощью CSS

Позиционирование осуществляется с помощью блока (`<div>`, например) с заданным параметром `display: flex;`, которые называются **flex-контейнером**. Все, что лежит в таких контейнерах называется **flex-элементом**.

# Параметры flex-контейнера

## justify-content

C помощью данного свойства можно выравнивать элементы по левому и правому краю, по центру, по краям, по всей ширине:

```
justify-content: flex-start; - По левому краю
justify-content: flex-end; - По правому краю
justify-content: center; - По центру
justify-content: space-between; - По краям
justify-content: space-around; - По ширине
```

## align-items

Данное свойство выравнивает flex-элементы по вертикали:

```
align-items: stretch; - растягивает flex-элемент по всей высоте контейнера (значение по умолчанию)
align-items: flex-start; - элементы прижимаются к верху контейнера
align-items: flex-end; - элементы прижимаются к низу контейнера
align-items: center; - элементы центрируются по вертикали
align-items: baseline; - элементы выравниваются по базовой линии, то есть если у flex-элементов не задана высота, то она вычисляется по его содержимому
align-items: stretch;- растягивает flex-элемент по всей высоте контейнера (значение по умолчанию)
align-items: flex-start; - элементы прижимаются к верху контейнера
align-items: flex-end; - элементы прижимаются к низу контейнера
align-items: center; - элементы центрируются по вертикали
align-items: baseline; - элементы выравниваются по базовой линии, то есть если у flex-элементов не задана высота, то она вычисляется по его содержимому
```

## flex-direction

Определяет как flex-элементы будут располагаться относительно друг друга:

```
flex-direction: row;- элементы располагаются слева на право относительно друг друга, данное значение идет по умолчани
flex-direction: row-reverse; - элементы расположены справа налево
flex-direction: column; - элементы расположены сверху вниз
flex-direction: column-reverse; - элементы расположены снизу вверх
```

## flex-wrap

Данное свойство отвечает за многострочность элементов и может принимать следующее значение:

```
flex-wrap: nowrap; - flex-элементы располагаются в одну строку без переносов (значение по умолчанию)
flex-wrap: wrap; - flex-элементы помещаются в несколько рядов слева направо, если не помещаются в строке
flex-wrap: wrap-reverse; - flex-элементы помещаются в обратном порядке справа налево причем перенос происходит вверх
```

## order

С помощью данного свойства flex-элементы переставляются в начало или конец:

```
.first_element{
	order:-1;
}
```

В примере выше значение -1 переставит элемент с классом `first_element` в самое начало, а 1 в конец.

## flex-basis

Указывает значение ширины flex-элемента в `px`, `em` или `%`:

```
.children {
	border: 1px solid #00f;
	flex-basis: 25%;
}

.main {
	height: 200px;
	display: flex;
}

.children {
	border: 1px solid #00f;
	flex-basis: 25%;
}

.main {
	height: 200px;
	display: flex;
}
```

Тут мы указали ширину для каждого дочернего элемента 25%.
Также есть свойство `flex`, c помощью которого можно указать элементу его базовую ширину и трансформацию:

```
.children {
	border: 1px solid #00f;
	flex: 0 1 25%;
}

.main {
	height: 200px;
	display: flex;
}

.children {
	border: 1px solid #00f;
	flex: 0 1 25%;
}

.main {
	height: 200px;
	display: flex;
}
```

В примере выше 0 - коэффициент увеличения ширины, 1 - коэффициент уменьшения ширины, 25% - ширина элемента.

# gap

Расстояние между элементами:
```
gap: 30px 50px;
```
Первое значение - расстояние между столбцами, второе - между строками
