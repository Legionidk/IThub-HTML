# Списки в HTML
Есть два вида списков в HTML:
- `<ul>` - unordered list (маркированный список)
- `<ol>` - ordered list (нумерованный список)
    - Используя атрибут `start` можно указать с какого числа начнется счет списка
    - Используя атрибут `type` можно указать формат счета списка
        - `1` - для цифр
        - `i` - для строчных Римских цифр (`I` для заглавных)
        - `a` - для строчных букв (`A` для заглавных)
# Элементы списков
Для того что-бы внутри списка текст был нумерован или маркирован, внутри самого тега списка (`<ul>` или `<ol>`) необходимо вписывать тег `<li>` (list item).
# Список описаний
Для создания списка описания используется тег `<dl>` (definition list), внутри него используются следующие теги:
- `<dt>` - термин
- `<dd>` - описание термина
