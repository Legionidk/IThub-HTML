# Списки в HTML
Есть два вида списков в HTML:
- `<ul>` - unordered list (маркированный список)
- `<ol>` - ordered list (нумерованный список)
# Элементы списков
Для того что-бы внутри списка текст был нумерован или маркирован, внутри самого тега списка (`<ul>` или `<ol>`) необходимо вписывать тег `<li>` (list item).
