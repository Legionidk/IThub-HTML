# Создание формы

Форма создается тегом `<form>`, а элементы формы тегом `<input>`. У тега `<input>` есть много применений благодаря атрибуту `type`, основные из них:

-   `text` - одностраничное поле ввода
-   `submit` - кнопка отправки формы
    -   Подобную кнопку можно так-же создать с помощью тега `<button>`, в котором можно указать атрибут `type="submit"`
-   `reset` - сброс формы
-   `radio` - radio кнопка
-   `checkbox` - чекбокс (клянусь)

# Подписи элементов - тег `<label>`

Добавить подпись для элемента формы можно с помощью парного тега `<label>`, у которого есть атрибут `for`, в котором указывается `id` элемента, к которому принадлежит `<label>`.

# Многострочные поля ввода - тег `<textarea>`

Тег `<textarea>` позволяет создать форму для ввода текста, которой можно задавать произвольную форму. У него есть некоторые атрибуты:

-   `rows` - задает изначальное количество рядов
-   `column` - задает изначальное количество столбцов

# Меню опций - тег `<select>`

С помощью тега `<select>` можно создать выпадающее меню с выбором заданных вариантов. Для добавления вариантов выбора необходимо использовать тег `<option>` внутри `<select>`.

У тега `<select>` есть атрибут `multiple`, который превращает выпадающее меню в список, в котором можно выбрать несколько вариантов используя L.CTRL.

У тега `<option>` также есть несколько атрибутов:

-   `selected` - вариант установлен как стандартное значение
-   `disabled` - вариант установлен как недоступный для выбора
