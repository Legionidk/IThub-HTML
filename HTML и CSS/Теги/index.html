Заголовок

У Заголовков (<h1>, h - header) есть несколько уровней, от первого до шестого. Уровень влияет на размер заголовка (заголовок, подзаголовок и так далее).

Параграф

Это параграф (<p>, p - paragraph). Используется для создания текста с форматированием.

Раздел

Это раздел (<div>, d - division). Он позволяет создавать блоки, которые объединяют несколько объектов на странице. Так-же у <div> есть особенность - он занимает всю ширину страницы.

Теги форматирования текста

Это тег <span> - контейнер, который используется в группировке фрагментов текста для стилизации с помощью CSS.
Это тег <i>, от слова italic (курсив).
Это тег <b>, от слова bold (жирный).
Это тег <s>, от слова strikethrough (зачеркнутый).
Это тег <u>, от слова underline (подчеркнутый).
Это тег <small>, который делает текст small :0.
Это тег <q>, от слова quote (цитата).
