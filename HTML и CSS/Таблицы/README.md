# Таблицы в HTML
В HTML таблицы создаются тегом `<table>`.
Для создания структуры необходимо использовать следующие теги:
- `<thead>`, `<tbody>`, `<tfoot>` - семантические теги, разделяющие заголовок, основание и "подвал" таблицы. Могут располагаться между тегами `<table>` и `<tr>`.
- `<tr>` - строка таблицы.
- `<td>` - ячейка таблицы, может находится только под тегом `<tr>`.
    - У тега есть два важных атрибута - `rowspan` и `colspan`, которые отвечают за объединение ячеек по столбцам и строкам соответственно.
- `<th>` - заголовочная ячейка, которая делает текст жирным и размещает текст по центру ячейки. Может находится только под тегом `<tr>`.
