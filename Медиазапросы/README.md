# Подключение медиазапросов

1. Подключение в `style.css`:
    ```
    @media screen and (min-width: 1280px) {
        <!-- Style code -->
    }
    ```
2. Подключение в `index.html`:
    ```
    <link rel="stylesheet" href="style.css" media="screen and (min-width: 1280px)"/>
    ```

# Аргументы медиазапросов

**Аргументы носителей**

1. `all` - все носители
2. `print` - печатающие устройства (принтер)
3. `projection` - проектор
4. `screen` - экран монитора
5. `tv` - телевизор

---

**Аргументы характеристик носителей**

1. `width` - ширина экрана носителя
    - Так-же есть параметры `min-width` и `max-width`
2. `height` - высота экрана носителя
    - Так-же есть параметры `min-height` и `max-height`
3. `aspect-ratio` - соотношение сторон экрана носителя (16:9, 4:3, 16:10)
4. `orientation` - ориентация экрана носителя (portrait, landscape)
5. `resolution` - разрешение экрана носителя (300dpi, 1280dpcm)

---

**Логические операторы**

Для связки разных переменных используется `and`, для отрицания `not`.
