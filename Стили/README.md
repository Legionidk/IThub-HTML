# Cascading StyleSheet

Для создания CSS стилей и встраивания их в HTML существует два способа:

1. Встраивание прямо в HTML файл с помощью тега `<style>` (под тегом `<head>`), в котором и описываются все стили.
2. Создание отдельного файла `style.css` и подключение его в HTML с помощью `<link>`.

# Разметка стилей

Разметка внутри `style.css` или `<style>` выглядит следующим образом:

```
<селектор> {
    <правило>: <значение>;
    <правило>: <значение>;
    <правило>: <значение>;
}
```

# Каскадность

В CSS `id`, `class` и сам `объект` имеют разный вес:

-   `id` > `class` > `object`

То есть при описании параметров для объектов сначала применятся всё, что задано по `id`, потом по `классу`, и только потом по `объекту`. В случае с объектом также есть правило, по которому будут заданы последние переданные параметры:

```
h2 {
	color: red;
}

h2 {
	color: blue;
}
```

`<h2>` будет иметь параметр `color: blue`, так как этот параметр был передан последним.

# Селекторы

Селекторы - это объекты HTML, для которых необходимо задать правила. Например, мы можем сделать цвет текста для тега `<p>` зеленым следующим образом:

```
p {
    color: green;
}
```

`green` можно заменить любым другим значением, например, rgb кодом.

В качестве селектора так-же могут выступать классы объектов или их id:

-   `.class` - использования класса в качестве селектора
-   `#id` - использование id в качестве селектора
-   `object.class` - использование объекта с определенным классом в качестве селектора
-   `object#id` - использование объекта с определенным id в качестве селектора.

# Вложенность

В CSS можно искать вложенные объекты:

-   `object_one object_two` - "ищи все `object_two` в `object_one` на любом уровне вложенности и задавай им параметры"
-   `object_one > object_two` - "ищи все `object_two` в `object_one` на первом уровне вложенности и задавай им параметры"
-   `object_one + object_two` - "ищи все `object_two` которые идут после `object_one` и задавай им параметры"
    -   **Важно:** эта конструкция ищет только один `object two` идущий после `object_one`
