# Рабочая среда - Apache
Управление серверами для удобной работы с **HTML** осуществляется через [Apache](https://httpd.apache.org/).
# Установка Apache
По дефолту **Apache** ожидает, что он будет установлен на диск `C:\`. При установке в другую папку необходимо поменять значение строчки `Define SRVROOT` в файле `*папка Apache*/conf/httpd.conf` на свой путь до программы.
# Автозапуск Apache
Для запуска сервисов Apache необходимо проделать следующие действия:
1. Перейти в папку с программой в консоли (`cd *папка Apache*`)
2. Прописать команду `httpd.exe -k install`
3. Запустить службу командой `net start apache2.4`
    - Для остановки службы необходимо воспользоваться командой `net stop apache2.4`
